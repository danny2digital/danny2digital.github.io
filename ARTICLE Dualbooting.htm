How to format a second HDD on an Old PC With Multiple HDD slots + Dualbooting

(Please note this article is a work in progress)

STEP 1:    Open Command Prompt As Administrator
 

Search for command prompt, right click and run as Administrator.

 

STEP 2:    Use Diskpart

Once command line is open, type 'diskpart' and press Enter.

STEP 3:    Type List Disk

The above command will open a Diskpart window. In this window, type 'list disk' and press Enter. It will list all the available drives.

STEP 4:    Select the Drive to Format

Now type 'select disk (disk number)' as shown above. Specify the drive number which needs to be formatted.

STEP 5:    Clean the Disk

In this step, type 'clean'. This command will permanent delete all files and folders, and successfully clean up the disk.

STEP 6:    Create Partition Primary

To make the drive again accessible, type 'create partition primary'.

STEP 7:    Format the Drive

Now format the drive with FAT or NTFS file system. Type 'format fs=ntfs' and press Enter.

STEP 8:    Assign a Drive Letter

To assign a drive letter, you can type 'assign'
Moreover, when you do diskpart and list volume command, simply 2. Type in diskpart

3. Type list disk to see a list of disks

4. Type select disk #  (where # is the disk you want)

5. Type detail disk to see partitions

6. Type select volume #  (where # is the volume you want)

7. Type assign letter=x  (where x is the drive letter)

Once that is assigned you should be good and able to dualboot once you reboot the computer with usb or cd

 

Dualbooting
 

This method will work for any OS (though i only tried it with windows 7 as the first OS so responsibility is on you for how you go about doing this)
Follow this guide and go to option 2: https://www.sevenforums.com/tutorials/8057-dual-boot-installation-windows-7-xp.html?ltr=D

Once the second OS (in this case XP is installed) make sure you download EasyBCD and have the required Net framework installed from an offline installer (tons can be found online). Now all you need to do is follow the remaining of their steps from 8 onwards in option 2.

The OS which gets booted will be managing the bootloader for the system Install EasyBCD (a free, easy to use software to manage your boot menu) You can download it from here: https://neosmart.net/EasyBCD/ Open EasyBCD and choose 'Add a new entry' Choose the type of your operating system, specify the partition letter, and Save changes. Restart your computer, and the next time you will see the bootmenu asking for which operating system to boot into. Note: If your bootmenu time is set to zero, the boot menu might not show up, so increase that time, it can also be done in EasyBCD.
